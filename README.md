# Portfolio
 Sayyeda Shifa — Portfolio Website

Overview

A modern, animated developer portfolio website built using pure HTML, CSS, and JavaScript,

designed to showcase projects, skills, and professional presence with a clean dark UI and smooth

micro■interactions.

Features

UI & Design

• Custom dark theme with lime accent color

• Glassmorphism navigation bar

• Animated hero section

• Scroll reveal animations

• Responsive layout

Sections Included

• Hero / Introduction

• Projects Showcase

• Skills & Tools

• FAQ & Contact Section

Tech Stack

Structure: HTML5

Styling: CSS3 (Custom Variables & Animations)

Fonts: Google Fonts (Syne, Inter, JetBrains Mono)

Interaction: Vanilla JavaScript

Deployment: GitHub Pages / Netlify / Vercel

Project Structure

portfolio/

■■■ index.html # Main website file

Getting Started

1. Clone the repository:

git clone https://github.com/your-username/portfolio.git

2. Open index.html in your browser. No installation required.

Deployment

You can deploy instantly using GitHub Pages, Netlify, or Vercel.

For GitHub Pages: push your repository, go to Settings → Pages, select branch main, and your

portfolio will be live
Customization

Edit CSS variables inside :root to change colors and update content directly inside index.html to

personalize name, projects, and social links.
